## 一、总体目标与 KPI 体系

### 1.1 愿景

> 将 su-memory LongMemEval 检索精度从当前 57.6% 提升至 85%+（中期），跨越 agentmemory V4 (96.2%) / Chronos (95.6%) 的行业顶级水平，同时保持能量中心（五行能量系统）作为核心差异化架构的完整性。

### 1.2 竞争对标矩阵

| 系统 | LongMemEval Accuracy | 核心范式 | 发布时间 |
|------|---------------------|----------|----------|
| **agentmemory V4** | **96.2%** | 6 路加权信号融合 + CrossEncoder + Claude Opus 4.6 | 2025 |
| **Chronos** | **95.6%** | 结构化事实 + 双索引 RRF + LLM 重排序 | 2025 |
| **Mastra OM** | **94.9%** | 图扩散 + 时序感知检索 | 2025 |
| **Hindsight** | **91.4%** | World/Experiences/Mental Models 三层架构 | 2024 |
| MemGPT | 78.3% | 操作系统式上下文管理 | 2024 |
| Mem0 | 72.5% | 向量 + 图谱混合记忆 | 2024 |
| Letta | 69.3% | Agent 自编辑记忆 | 2024 |
| Zep | 66.1% | 时序图谱持久记忆 | 2024 |
| **su-memory (当前)** | **57.6%** | 语义+TF-IDF 双路 + SpacetimeIndex + 能量中心 | 2025 |
| GPT-4 Turbo (裸) | 53.5% | 全量上下文窗口 | 2024 |

### 1.3 KPI 体系架构

```
                        ┌──────────────────────────────────────┐
                        │      检索能力成熟度 L1 → L4          │
                        │    （终极目标：LongMemEval ≥ 85%）    │
                        └──────────────────┬───────────────────┘
                                           │
         ┌─────────────────────────────────┼──────────────────────────────┐
         │                                 │                              │
  ┌──────▼──────┐                  ┌──────▼──────┐              ┌───────▼──────┐
  │  召回 KPI   │                  │  精度 KPI   │              │  工程 KPI    │
  │             │                  │             │              │              │
  │ Recall@10   │                  │ Accuracy    │              │ JSON成功率   │
  │ 双路召回覆盖 │                 │ 六维度分数  │              │ 事件提取可回放│
  │ 信号丰富度  │                  │ 放弃识别 P/R│              │ 消融可追溯   │
  └─────────────┘                  └─────────────┘              └──────────────┘
```

### 1.4 核心原则

| 原则 | 说明 |
|------|------|
| **消融可追溯** | 每个 Phase 改动必须可独立开关（feature flag），能力跑完整 500 题消融验证 |
| **能量中心完整性** | 不破坏五行生克核心代码（7000+ 行 13 模块），只在入口层（`lite_pro.py`）改接线 |
| **竞品对齐** | 每个 Phase 的检索范式必须有竞品依据，不做无参照的"拍脑袋"优化 |
| **回归防护** | 每次改动后跑完整 500 题，分维度报告，与上一版本对比 |
