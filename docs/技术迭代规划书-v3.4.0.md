## 〇、总纲：能量中心为核，四层量化为翼

### 核心哲学

su-memory SDK 的灵魂是**能量中心（Energy Center）**——由 `EnergyBus`（能量总线）、`EnergyCore`（能量核心引擎）、`EnergyRelations`（Energy Types enhance/suppress体系）组成的语义-因果-时空-可信四维能力中枢。它定义了系统"是什么"：five-element enhance/suppress cycle、Trigram Patterns temporal-spatial mapping、干支时间编码、能量传播与平衡制化。

v3.3.0 的因果引擎停留在**语法层**——用关键词模式匹配检测显式因果句，置信度硬编码 0.6–0.95。这离"真正的因果理解"还差一层：**数学可验证性**。

v3.4.0 的使命不是替换能量中心，而是为它装上**量化眼镜**——用四层数学框架（傅里叶频域视角、高斯统计检验、贝叶斯不确定性量化、因果概率 do-operator）将能量中心的直觉判断转化为**可计算、可验证、可解释的概率因果模型**。

```
能量中心 (Energy Center) — 核心骨架
    │ 语义维度 (semantic)  ●  因果维度 (causal)  ●  时空维度 (spacetime)
    │ 生成维度 (generative) ●  信任维度 (trust)
    │
    ├── 傅里叶层 (Fourier)     → 频域视角：滤除周期噪声，提取因果信号
    ├── 高斯层 (Gaussian)      → 统计检验：偏相关系数，隐藏因果发现
    ├── 贝叶斯层 (Bayesian)    → 不确定性量化：后验概率，置信区间
    └── 因果概率层 (Causal)    → do-operator：P(Y|do(X)) 因果效应估计
```

## 一、能量中心深度剖析：为什么它是因果建模的天然骨架

### 1.1 five-element enhance/suppress cycle = 5 状态离散周期信号

```python
# _energy_relations.py — 这是因果建模的先天结构
ENERGY_ENHANCE  = {"wood":"fire", "fire":"earth", "earth":"metal", "metal":"water", "water":"wood"}  # 生循环 period=5
ENERGY_SUPPRESS = {"wood":"earth","earth":"water","water":"fire","fire":"metal","metal":"wood"}      # 克循环 period=5
```

任一元素的强度随时间演化是**5 周期离散信号**。以木为例：

```
寅月(旺 WANG=1.2) → 传播到火(生) → 传播到土(生) → 传播到金(生) → 传播到水(生) → 回木
每步 decay_rate=0.85, 5 步后强度 = 1.2 × 0.85^5 ≈ 0.53
```

这条轨迹 `[1.2, 1.02, 0.87, 0.74, 0.63, 0.53, ...]` 正是 FFT 的标准输入——**5 个采样点的离散傅里叶变换恰好分解为一个 DC 分量 + 一个基频 (f=1/5) + 高次谐波**。

`EnergyBus.propagate_energy()` 已记录 `_signal_history` 和 `_balance_history`——这些就是频谱分析的原始数据。

### 1.3 生克 DAG = 因果发现的先验锚定

当前 `BayesianNetwork` 需要手动 `add_causal_link(cause, effect)`。但五元素系统天然提供了一张**预建因果图**：

```
10 条有向边:
  生循环 (5 条, enhance): 木→火, 火→土, 土→金, 金→水, 水→木
  克循环 (5 条, suppress):木→土, 土→水, 水→火, 火→金, 金→木
```

这张图不能直接作为统计因果答案，但它提供了**贝叶斯先验**——偏相关系数计算时，生克边的阈值可以适当放宽（先验信任），非生克边需要更严格的 p-value。

## 二、四层量化框架 · 分层详述

### 第一层：傅里叶频域因果 (FourierCausal) — 🆕 全新

**解决的问题**: 周期性波动 vs 真实因果事件的区分

#### 数学原理

五元素系统是离散周期信号，FFT 将其分解为：

\[
X[k] = \sum_{n=0}^{4} x[n] \cdot e^{-j 2\pi k n / 5}, \quad k = 0,1,2,3,4
\]

- **DC 分量 (k=0)**: 总能量水平
- **基频 (k=1/5)**: 生循环正常代谢节奏
- **高次谐波 (k=2/5)**: 克循环制约活动
- **残余高频**: 异常波动 → **因果事件候选**

#### 三个注入点

| 注入点 | 能量总线接口 | FourierCausal 处理 | 产出 |
|--------|------------|-------------------|------|
| **注入 ①** 能量传播谱分析 | `EnergyBus._signal_history` | FFT 每条传播路径的强度序列 | 区分正常生克传播 vs 因果干预 |
| **注入 ②** 能量平衡频谱诊断 | `EnergyCore._calculate_energy_balance()` | 五维强度序列的联合频谱 | 高频分量占比 → 系统失衡度 |
| **注入 ③** 周期混淆因子过滤 | `GaussianDAG.partial_correlation()` 前处理 | 滤除共享周期分量后的偏相关 | 消除"冰淇淋↔溺水"型假因果 |

#### 核心算法：频谱混淆因子检测

```
输入: 记忆 A 的强度序列 x[n], 记忆 B 的强度序列 y[n]
步骤:
  1. FFT: X[f] = FFT(x), Y[f] = FFT(y)
  2. 相干性: C[f] = |X[f]·Y[f]*|² / (|X[f]|²·|Y[f]|²)
  3. 周期分量: f_periodic = {f | C[f] > 0.8}  (大概率共享周期)
  4. 滤除: x_clean = IFFT(X[f] - X[f_periodic])
  5. 重算偏相关: ρ = pearsonr(x_clean, y_clean)
  → 如果清洗后 ρ ≈ 0，说明原相关是周期混淆，非因果
```

#### 与能量中心的契合度

| 维度 | 评分 | 说明 |
|------|:--:|------|
| 信号输入 | 🟢 | `_signal_history` 天然是时间序列 |
| 周期匹配 | 🟢 | 五元素周期=5 完美对应 5 点 FFT |
| 噪声来源 | 🟢 | `decay_rate=0.85` 定义正常衰减，异常值即噪声 |
| 代码侵入 | 🟢 | 零侵入——对已有信号历史做后处理分析 |

### 第二层：高斯统计因果 (GaussianDAG) — 升级自 v1.0 版规划

**解决的问题**: 隐藏因果边发现（无关键词标记的因果关系）

#### 数学原理

在高斯假设下（误差 \( \varepsilon \sim \mathcal{N}(0, \sigma^2) \)），条件独立性等价于偏相关系数为零：

\[
\rho_{XY|Z} = \frac{\rho_{XY} - \rho_{XZ}\,\rho_{YZ}}{\sqrt{(1-\rho_{XZ}^2)(1-\rho_{YZ}^2)}}
\]

\[
\rho_{XY|Z} = 0 \Longleftrightarrow X \perp\!\!\!\perp Y \mid Z
\]

#### 与能量中心的交叉验证（v1.0 版规划中缺失的关键增强）

原 v1.0 版 `GaussianDAG` 是纯粹的统计扫描。加入能量中心先验后，升级为**三重交叉验证**：

```
偏相关发现 (pearsonr)  ──┬── p < 0.05 ? → 候选因果边
                          │
五元素生克关系 ───────────┴── 生/克关系匹配?
                          │
                          ├─ 统计+能量 都指向 → 高信因果 ✅ (confidence × 1.2)
                          ├─ 统计有、能量无 → 新发现因果 🆕 (标记为 novel)
                          ├─ 统计无、能量有 → 潜在抑制 ⚠️ (克关系压制了相关性)
                          └─ 两者都无   → 无因果
```

#### 关键增强：能量先验加权

```python
def causal_confidence_with_energy_prior(
    rho: float, p_value: float, 
    energy_relation: Optional[RelationType]
) -> float:
    """
    在偏相关系数基础上融入能量先验。
    
    - 生关系 (ENHANCE) + 偏相关显著: 置信度 × 1.2
    - 克关系 (SUPPRESS) + 偏相关边缘: 置信度 × 0.8 (惩罚)
    - 无能量关系 + 偏相关显著: 保持原置信度
    - 生关系但偏相关不显著: 标记为"潜在因果，需更多数据"
    """
    base_conf = min(abs(rho), 0.95)
    
    if energy_relation == RelationType.ENHANCE:
        if p_value < 0.05:
            return min(base_conf * 1.2, 0.98)  # 上限 0.98
        else:
            return base_conf * 0.8  # 能量支持但统计不显著，降权
    elif energy_relation == RelationType.SUPPRESS:
        return base_conf * 0.8  # 克关系天然弱相关
    return base_conf
```

### 第三层：贝叶斯不确定性量化 (BayesianCausal) — 升级自 Beta 体系

**解决的问题**: 从点估计到概率分布，量化因果推断的可靠程度

#### 从 Beta-Binomial 到 Gaussian-Gaussian

v3.3.0 的 `BayesianEngine` 已实现完整的 Beta-Binomial 共轭系统（794 行，`BetaDistribution` 含均值/方差/Credible Interval/假设检验）。v3.4.0 新增 **Gaussian-Gaussian 共轭**，因为因果效应的连续值（偏相关系数、效应大小）更适合高斯建模：

| 类 | 用途 | 共轭族 | 适用场景 |
|----|------|--------|---------|
| `BetaDistribution` (已有) | 二项概率信念 | Beta-Binomial | 事件是否会发生 |
| **`GaussianDistribution` (新增)** | 连续效应量 | Normal-Normal | 效应有多大 🤌 |
| **`CausalPosterior` (新增)** | 因果效应后验 | Gaussian + 生克先验 | P(Y\|do(X)) |

### 第四层：因果概率 do-Operator (CausalProbability) — 终极目标

**解决的问题**: 区分 P(Y|X) 和 P(Y|do(X))，回答"干预"问题

#### 数学框架

\[
\text{相关: } P(Y|X) \quad \text{—— 观测到的统计关联}
\]
\[
\text{因果: } P(Y|\text{do}(X)) \quad \text{—— 主动干预 X 后 Y 的分布}
\]

v3.4.0 用**后门调整公式**实现 do-operator：

\[
P(Y|\text{do}(X)) = \sum_{Z} P(Y|X, Z) \cdot P(Z)
\]

其中 Z 是满足后门准则的变量集合——在五元素系统中，Z 就是生克链上的中间节点。

#### 落地路径（分阶段）

```
M1: GaussianDAG → 偏相关发现 + 能量先验交叉验证
    └─ 产出: 候选因果边 + 方向 + p-value

M2: FourierCausal → 频域混淆因子过滤
    └─ 产出: 清洗后的因果边（排除周期伪相关）

M3: BayesianCausal → 后验概率 + 贝叶斯因子
    └─ 产出: P(effect_exists | data) + 95% CI

M4: CausalProbability → do-operator 干预估计 
    └─ 产出: P(Y|do(X)) = Σ_Z P(Y|X,Z)·P(Z)
```

**v3.4.0 交付 M1 + M2 + M3**，M4 (do-operator) 预留 v3.5.0——需要更完整的 DAG 结构。

#### 子任务 4.1.1：新建 `_spectral_causal.py`

**文件**: `src/su_memory/sdk/_spectral_causal.py`（新建）  
**内容**: `GaussianDAG` + `FourierCausal` + `BayesianCausal` 三个类

```python
import numpy as np
from scipy.stats import pearsonr, norm
from typing import List, Dict, Tuple, Optional

class GaussianDAG:
    """
    基于高斯假设的偏相关因果发现。
    
    核心公式:
    ρ_{XY|Z} = (ρ_{XY} - ρ_{XZ}·ρ_{YZ}) / sqrt((1-ρ_{XZ}²)(1-ρ_{YZ}²))
    
    与能量中心对接:
    - _energy_boost(): 根据生克关系调整置信度
    - 生 (ENHANCE): 能量支持 → 置信度 × 1.2
    - 克 (SUPPRESS): 能量抑制 → 置信度 × 0.8
    """
    
    def __init__(self, memories: List[Dict], tfidf_index: Dict[str, set],
                 energy_bus=None):
        self.memories = memories
        self._index = tfidf_index
        self._energy_bus = energy_bus  # 可选的能量总线引用
        
    def partial_correlation(self, x, y, z) -> Tuple[float, float]:
        """偏相关系数 + Fisher z-transform p-value"""
        r_xy, _ = pearsonr(x, y)
        r_xz, _ = pearsonr(x, z)
        r_yz, _ = pearsonr(y, z)
        num = r_xy - r_xz * r_yz
        denom = np.sqrt((1 - r_xz**2) * (1 - r_yz**2))
        if abs(denom) < 1e-10:
            return (0.0, 1.0)
        rho = num / denom
        n = len(x)
        z_stat = 0.5 * np.log((1 + rho) / (1 - rho)) * np.sqrt(n - 3)
        p_value = 2 * (1 - norm.cdf(abs(z_stat)))
        return (rho, p_value)
    
    def energy_prior_boost(self, mem_a: Dict, mem_b: Dict, 
                           base_conf: float) -> Tuple[float, str]:
        """
        能量先验交叉验证。
        
        Returns:
            (adjusted_confidence, verdict)
            verdict: "confirmed" | "novel" | "suppressed" | "none"
        """
        if self._energy_bus is None:
            return (base_conf, "none")
        # 从记忆中提取能量类型
        etype_a = mem_a.get("energy_type", "earth")

@dataclass
class GaussianDistribution:
    """
    高斯分布 — 连续因果效应的共轭先验。
    
    平行于 BetaDistribution (已有)，用于建模连续量:
    - 偏相关系数的后验分布
    - 因果效应大小的后验分布
    - 干预效应的后验分布
    
    Normal-Normal 共轭:
    先验: N(μ₀, σ₀²)
    似然: N(μ, σ²/n)  (n 个观测的样本均值分布)
    后验: N(μₚ, σₚ²) 其中:
      μₚ = (μ₀/σ₀² + n·x̄/σ²) / (1/σ₀² + n/σ²)
      σₚ² = 1 / (1/σ₀² + n/σ²)
    """
    mu: float = 0.0       # 均值
    sigma: float = 1.0    # 标准差
    n_observations: int = 0
    
    @property
    def mean(self) -> float:
        return self.mu
    
    @property
    def variance(self) -> float:
        return self.sigma ** 2
    
    @property
    def std(self) -> float:
        return self.sigma
    
    def credible_interval(self, prob=0.95) -> Tuple[float, float]:
        """高斯 HDPI ≈ μ ± z·σ"""
        z = {0.90: 1.645, 0.95: 1.96, 0.99: 2.576}.get(prob, 1.96)
        return (self.mu - z * self.sigma, self.mu + z * self.sigma)
    
    def update(self, sample_mean: float, sample_std: float, n: int):
        """Normal-Normal 共轭更新"""
        prior_precision = 1.0 / (self.sigma ** 2)
        likelihood_precision = n / (sample_std ** 2) if sample_std > 0 else 0
        
        post_precision = prior_precision + likelihood_precision
        self.mu = ((self.mu * prior_precision + sample_mean * likelihood_precision) 
                   / post_precision)
        self.sigma = np.sqrt(1.0 / post_precision)
        self.n_observations += n
```

```python
class BayesianCausal:
    """
    贝叶斯因果推断 — 能量先验增强版。
    """
    
    def __init__(self, energy_bus=None):
        self._energy_bus = energy_bus
        self._causal_posteriors: Dict[str, GaussianDistribution] = {}
    
    def causal_hypothesis_test(
        self,
        edge_id: str,
        rho: float,        # 观测到的偏相关系数
        n_samples: int,    # 样本量
        energy_relation: Optional[RelationType] = None
    ) -> Dict:
        """
        贝叶斯因果假设检验。
        
        H₀: ρ = 0 (无因果效应)
        H₁: ρ ≠ 0 (有因果效应)
        
        先验融入:
        - 生关系 → 先验 μ₀ > 0 (正效应预期)
        - 克关系 → 先验 μ₀ ≈ 0 (保守，克天然弱相关)
        - 无关系 → 先验 μ₀ = 0 (无信息先验)
        """
        # 先验选择
        if energy_relation == RelationType.ENHANCE:
            prior = GaussianDistribution(mu=0.3, sigma=0.5)
        elif energy_relation == RelationType.SUPPRESS:
            prior = GaussianDistribution(mu=0.0, sigma=0.3)  # 更窄更保守
        else:
            prior = GaussianDistribution(mu=0.0, sigma=1.0)
        
        # 后验更新 (Normal-Normal)
        prior.update(sample_mean=rho, sample_std=1/np.sqrt(n_samples), n=n_samples)
        
        # 贝叶斯因子: BF₁₀ = P(D|H₁)/P(D|H₀)
        # Savage-Dickey: BF₁₀ = prior_density(0) / posterior_density(0)
        from scipy.stats import norm as norm_dist
        prior_at_zero = norm_dist.pdf(0, prior.mu, prior.sigma)
        post_at_zero = norm_dist.pdf(0, prior.mu, prior.sigma)  # 更新后的 prior
        
        if post_at_zero > 0:
            bf = prior_at_zero / post_at_zero
        else:
            bf = float('inf')
        
        ci = prior.credible_interval(0.95)
        
        return {
            "posterior_mean": prior.mu,
            "posterior_std": prior.sigma,
            "credible_interval_95": ci,
            "bayes_factor": bf,
            "conclusion": (
                "有因果效应" if ci[0] > 0 
                else "无因果效应" if ci[1] < 0
                else "证据不足"
            ),
            "energy_prior_used": energy_relation is not None,
        }
```
