#### P0-1: max_memories限制失效

**根因**: `add()` 方法在第225行检查限制，但 `_load()` 持久化加载时未裁剪超出部分。

**修复文件**: `src/su_memory/sdk/lite.py`

```python
# 第189行 add()方法，在添加前先淘汰
def add(self, content: str, metadata: Dict[str, Any] = None) -> str:
    # 添加前先淘汰，确保不超过限制
    while len(self._memories) >= self.max_memories:
        self._evict_oldest()
    # ... 原有逻辑

# _load()方法加载后裁剪
def _trim_to_max(self) -> None:
    """裁剪到最大限制"""
    while len(self._memories) > self.max_memories:
        self._evict_oldest()
```

#### P0-2: SQLite向量维度不匹配

**根因**: `sqlite_backend.py:537` 传入128维向量，但存储配置为256维。

**修复文件**: `src/su_memory/storage/sqlite_backend.py`

```python
# 第75行 SQLiteBackend类
class SQLiteBackend:
    def __init__(
        self,
        db_path: str,
        embedding_dim: int = 384,  # 统一默认维度
        timeout: float = 30.0
    ):
        self._embedding_dim = embedding_dim
        self._timeout = timeout
    
    def _ensure_embedding_dim(self, embedding: List[float]) -> np.ndarray:
        """确保向量维度一致"""
        arr = np.array(embedding, dtype=np.float32)
        if len(arr) != self._embedding_dim:
            if len(arr) < self._embedding_dim:
                arr = np.pad(arr, (0, self._embedding_dim - len(arr)))
            else:
                arr = arr[:self._embedding_dim]
        return arr / (np.linalg.norm(arr) + 1e-8)
```

#### P0-3: SQLite数据库锁定

**根因**: 缺少连接超时设置和事务隔离。

**修复**: 添加超时和重试机制

```python
def _get_connection(self) -> sqlite3.Connection:
    conn = sqlite3.connect(
        self._db_path,
        timeout=self._timeout,  # 30s超时
        isolation_level='DEFERRED'
    )
    conn.execute("PRAGMA busy_timeout = 30000")
    conn.execute("PRAGMA journal_mode = WAL")
    return conn

def _execute_with_retry(self, func, max_retries=3):
    """带重试的执行"""
    for attempt in range(max_retries):
        try:
            return func()
        except sqlite3.OperationalError as e:
            if "locked" in str(e) and attempt < max_retries - 1:
                time.sleep(0.1 * (attempt + 1))
                continue
            raise
```

#### P0-4: DataExporter导出失败

**根因**: 与SQLiteBackend共享连接导致锁定。

**修复**: 使用只读连接和独立会话

```python
# exporter.py
def _get_readonly_connection(self) -> sqlite3.Connection:
```

#### P1-2: 查询性能优化

**当前问题**: 线性扫描导致10K数据延迟暴增

**优化方案**:
```python
# lite.py query()方法优化
def query(self, query: str, top_k: int = 5) -> List[Dict[str, Any]]:
    cache_key = (query, top_k)
    if cache_key in self._query_cache:
        self._cache_hits += 1
        self._query_cache.move_to_end(cache_key)
        return self._query_cache[cache_key].copy()
    
    query_keywords = self._extract_keywords(query)
    if not query_keywords or not self._memories:
        return []
    
    # 使用倒排索引快速过滤候选集
    candidate_ids = None
    for keyword in query_keywords:
        if keyword in self._index:
            if candidate_ids is None:
                candidate_ids = self._index[keyword].copy()
            else:
                candidate_ids &= self._index[keyword]  # 交集
    
    # 仅对候选集评分
    scores = {}
    if candidate_ids:
        for memory_id in candidate_ids:
            scores[memory_id] = self._calculate_tfidf(memory_id, query_keywords)
```
