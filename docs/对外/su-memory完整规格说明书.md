# su-memory 企业级AI记忆中台
## 完整规格说明书

---

## 一、产品定位

**su-memory** 是企业级AI长期记忆中台，为私有化大模型提供人类级别的记忆能力。

让AI系统能够记住、理解和运用长期积累的信息，实现真正的个性化服务和连贯推理。
