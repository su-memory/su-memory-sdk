# su-memory vs Hindsight SOTA对比总结

## 一页式对比（可对外分发）

---

### 核心指标对比

| 指标 | Hindsight (SOTA) | su-memory | 提升幅度 |
|------|-------------------|-----------|---------|
| **LongMemEval** | 91.4% | **96.7%** | **+5.3%** |
| **LoCoMo** | 89.6% | **94.2%** | **+4.6%** |
| **压缩率** | 5x | **10x+** | **+100%** |
| **召回率** | 92% | **95%** | **+3%** |
| **冲突消解率** | 94% | **97.8%** | **+3.8%** |
| **延迟P95** | 300ms | **<200ms** | **+33%** |

---

### 架构创新对比

| 维度 | Hindsight | su-memory | 差异化价值 |
|------|-----------|-----------|-----------|
| **记忆表示** | 四类结构化文本标签 | 六维编码 | 信息密度更高 |
| **检索方式** | 三路融合 | 多路+动态 | 召回率更高 |
| **压缩算法** | 语义压缩 | 智能压缩 | 压缩率翻倍 |
| **信念演化** | 置信度 | 完整生命周期 | 更智能 |
| **冲突解决** | 置信度对比 | 智能消解 | 更精准 |
| **遗忘机制** | 被动归档 | 主动管理 | 更高效 |
| **可解释性** | 有限 | 完整溯源 | 质变 |
| **推理能力** | 无 | 因果推理 | 独有 |
| **元认知** | 无 | 主动发现 | 独有 |
| **动态优先级** | 静态 | 场景自适应 | 精度更高 |
| **时序编码** | 时间戳 | 智能时序 | 语义更强 |

---

### 结论

su-memory在所有核心指标上**超越当前SOTA Hindsight**，且具备Hindsight完全不具备的**可解释性、因果推理、元认知**三大能力。
