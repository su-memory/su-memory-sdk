# su-memory SDK 产品介绍

> 易学AI记忆引擎 · 本地优先 · 因果推理 · 可解释记忆

---

## 🎯 一句话介绍

**su-memory SDK** 是基于因果推理和时序感知的AI记忆引擎，让大语言模型具备持久记忆和推理能力。

---

## 💡 核心痛点

| 痛点 | 解决方案 |
|------|----------|
| LLM缺乏长期记忆 | 本地向量存储，持久化记忆 |
| 检索结果不可解释 | 推理链追溯，置信度分解 |
| 缺乏因果推理能力 | MemoryGraph因果图谱 |
| 时序信息丢失 | 时序编码系统 |
| 数据隐私泄露 | 本地优先，无需云端 |

---

## ✨ 核心功能

### 1. 语义向量检索
```
• Ollama bge-m3 本地向量模型
• 1024维中文语义编码
• RRF混合检索融合
```

### 2. 因果推理
```
• MemoryGraph因果图谱
• BFS多跳遍历
• 因果类型检测 (cause/condition/result)
```

### 3. 时序感知
```
• 时序编码系统
• 时效性指数衰减
• 时序编码计算
```

### 4. 可解释性
```
• 推理链可视化
• 置信度因素分解
• 自然语言解释生成
```

## 🚀 快速开始

```bash
pip install su-memory
```

```python
from su_memory.sdk import SuMemoryLitePro

# 创建客户端
pro = SuMemoryLitePro(embedding_backend='ollama')

# 添加记忆
pro.add("如果努力学习，成绩会提高")
pro.add("成绩提高了会获得奖学金")

# 建立因果链
pro.link_memories(pro._memories[-2].id, pro._memories[-1].id)

# 多跳推理
results = pro.query_multihop("学习", max_hops=3)

# 时序预测
predictions = pro.predict(query="项目活动")

# 可解释查询
explanation = pro.explain_query("学习", results)
```
