## 一、项目概述

### 1.1 项目定位

**su-memory — 周易AI记忆引擎**

用《周易》思想重构 AI 记忆系统的第一性原理，以六十四卦为记忆编码体系、五行生克为能量动力学、八卦类象为语义拓扑、天干地支为时空坐标，打造**可解释、可推理、医用级**的私有化 AI 记忆服务。

**核心差异化**：
- 记忆 = 六爻卦象 + 五行能量 + 干支时空 + 因果链 + 原文
- 每条记忆皆可解读其卦象来源、能量状态、时空坐标
- 全面超越 Mem0/Zep 的记忆深度

### 1.2 核心目标

| 层级 | 目标 |
|------|------|
| **商业目标** | 全面超越 Mem0/Zep 商用版，主导政企/医疗/私域AI市场 |
| **技术目标** | 用周易框架解决"信息密度低、语义扁平、无法推理"的核心问题 |
| **差异化目标** | 成为第一个"可解释记忆系统"——每条记忆的来源、状态、关系皆可解读 |

### 2.3 记忆表示模型

```python
class SuMemoryItem:
    # === 易理核心（六维编码）===
    hexagram: int              # 0-63，64卦之一
    hexagram_details: dict       # 卦名、卦辞、变爻、互卦、综卦、错卦
    bagua: str                 # 八卦（8类）
    bagua_energy: float         # 卦气能量 0.0-2.0
    wuxing: str                # 五行（木火土金水）
    wuxing_energy: float        # 五行能量
    tian_gan: str              # 天干
    di_zhi: str                # 地支
    jiazi_index: int           # 六十甲子索引 0-59

    # === 标准字段 ===
    memory_id: str
    content: str                # 原文
    embedding: List[float]      # 向量
    causal_parent: List[str]    # 因果父节点
    causal_children: List[str]  # 因果子节点
    timestamp: float            # 时间戳
    tenant_id: str              # 租户ID
```

## 五、易学四位一体理论体系

### 5.1 八卦语义分类

| 八卦 | 符号 | 五行 | 核心语义 | AI记忆映射 |
|------|------|------|---------|-----------|
| 乾 | ☰ | 金 | 刚健/积极/上升 | 高价值/重要/活跃记忆 |
| 坤 | ☷ | 土 | 柔顺/承载/稳定 | 稳定/持久/根基性记忆 |
| 离 | ☲ | 火 | 明亮/发散/连接 | 知识/事实/关联网络 |
| 坎 | ☵ | 水 | 陷入/危险/隐匿 | 问题/风险/隐藏事项 |
| 震 | ☳ | 木 | 震动/变化/触发 | 触发器/事件/变化 |
| 巽 | ☴ | 木 | 进入/渗透/扩散 | 渗透/扩散/影响力 |
| 艮 | ☶ | 土 | 停止/阻碍/守候 | 障碍/暂停/等待 |
| 兑 | ☱ | 金 | 喜悦/交换/协议 | 协议/交换/正向反馈 |

### 5.2 五行生克

```
相生（木→火→土→金→水→木）：
木生火 → 火生土 → 土生金 → 金生水 → 水生木

相克（木克土→土克水→水克火→火克金→金克木）：
木克土 → 土克水 → 水克火 → 火克金 → 金克木
```

### 5.3 三易法则

| 法则 | 定义 | 在记忆系统中的映射 |
|------|------|-----------------|
| **不易** | 恒定规律 | 五行生克、阴阳匹配、层级映射 |
| **变易** | 动态变化 | 五行旺衰、干支合冲、卦爻动静 |
| **简易** | 核心精髓 | 以简驭繁、抓核心气机 |
