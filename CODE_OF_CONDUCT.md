# 社区行为准则

## 我们的承诺

我们致力于为每个人提供开放、包容的环境。无论经验水平如何，我们都欢迎所有贡献者。

## 我们的标准

### 积极贡献的行为

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注社区共同利益
- 对其他社区成员表示同情

### 不可接受的行为

- 使用涉及性骚扰的语言或图像
- 进行人身攻击或发表侮辱性评论
- 公开或私下骚扰
- 未经许可发布他人私人信息
- 其他不专业的行为

## 我们的责任

项目维护者有责任明确解释可接受行为的标准，并对不可接受行为采取适当、公平的纠正措施。

## 执行

发现不可接受的行为，请通过以下方式联系项目团队：

**邮箱**：sandysu737@gmail.com

我们会认真对待每一份报告，并进行保密调查。

## 处理方针

1. 项目团队将审阅并调查所有投诉
2. 调查结果是保密的，并与相关人员适度沟通
3. 项目团队有义务对举报人保密
4. 严重违反者将被永久禁止参与项目

## 适用范围

本行为准则适用于所有公共空间，也适用于代表项目或社区的个人。

## 处罚

任何不可接受的行为都可能导致：
- 书面警告
- 暂时禁止参与
- 永久禁止参与
